# GameSubpoter

웹프로그래밍 기말 프로젝트용 게임 리뷰 웹사이트입니다.

## 프로젝트 개요

GameSubpoter는 게임 구매 전 사용자 리뷰, 별점, 한줄평을 확인하고 직접 리뷰를 작성할 수 있는 리뷰 서비스입니다. 흰색, 검은색, 스페이스 그레이 중심의 차분한 카드형 구성을 참고했습니다.

## 주요 기능

- 빈 상태의 게임 리뷰 목록 UI
- 리뷰 등록 후 카드형 목록 표시
- 리뷰가 여러 개일 때 자동 전환 및 화살표 버튼 슬라이더
- 자세히 보기 모달
- 게임 이름, 장르, 플랫폼 기반 검색 및 필터
- 리뷰 작성 폼
- 게임 사진 업로드 미리보기
- 로그인 및 회원가입 UI
- 마이페이지 UI

## 사용 기술

- HTML
- CSS
- JavaScript

## 파일 구조

- `index.html`: 웹페이지 구조
- `styles.css`: 전체 디자인과 반응형 스타일
- `script.js`: 리뷰 슬라이더, 검색, 상세 보기, 업로드 미리보기 동작
- `assets/banner/`: 메인 배너에 사용할 게임 포스터 이미지

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다.

메인 배너 이미지는 `assets/banner/README.md`에 적힌 파일명으로 넣으면 하나의 포스터형 콜라주처럼 표시됩니다.

## 향후 확장 계획

- 리뷰 데이터 DB 저장
- 로그인 사용자별 리뷰 작성/수정/삭제 권한
- 댓글 기능
- 찜 목록 저장
- 외부 게임 API 연동
- 취향 기반 게임 추천 기능

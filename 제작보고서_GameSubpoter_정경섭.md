# 제작보고서: GameSubpoter

## 1. 프로젝트 정의

GameSubpoter는 게임 리뷰를 카드 형태로 제공하고, 사용자가 게임별 별점과 한줄평, 상세 리뷰를 확인하거나 작성할 수 있는 웹사이트입니다. 게임을 구매하기 전 실제 사용자 평가를 확인하고 싶은 사용자를 주요 대상으로 합니다.

## 2. 구현 내용

메인 화면 상단에는 사이트 이름과 주요 메뉴 버튼을 배치했습니다. 중심 영역에는 게임 이미지, 게임 이름, 별점, 한줄평, 자세히 보기 버튼을 포함한 카드형 리뷰 UI를 구현했습니다. 리뷰 카드는 일정 시간마다 자동 전환되며, 좌우 화살표 버튼으로 직접 이동할 수도 있습니다.

리뷰 작성 영역에는 게임 이름, 장르, 플랫폼, 게임 사진 업로드, 별점, 한줄평, 상세 리뷰, 등록 버튼을 배치했습니다. 업로드한 이미지는 즉시 미리보기로 표시됩니다.

게임 검색 영역에서는 키워드 검색과 장르 필터를 통해 리뷰 카드를 빠르게 찾을 수 있도록 했습니다. 로그인 및 회원가입 UI와 마이페이지 UI도 함께 구성했습니다.

## 3. 디자인 방향

흰색, 검은색, 스페이스 그레이 중심의 차분한 색상과 깔끔한 상단 내비게이션, 넓은 히어로 영역을 사용했습니다. 메인 배너는 여러 게임 이미지를 자연스럽게 배치한 포스터형 콜라주처럼 보이도록 설계했습니다. 리뷰 목록은 처음에는 비어 있도록 구성했고, 사용자가 리뷰를 등록하면 카드형 UI로 표시되도록 구현했습니다.

## 4. 사용 기술

- HTML
- CSS
- JavaScript

## 5. AI 활용

생성형 AI Chat gpt를 활용했습니다.

처음 웹사이트 디자인은 유명 게임 플랫폼인 'Steam'을 참고하였으나, 대부분이 검은색인 해당 사이트 특성 상
시각적으로 알아보기 어렵다고 느껴 수정하였습니다. AI의 도움을 받아 'Apple'의 흰색, 검은색, 스페이스 그레이를 참고하여 조금 더 세련된 디자인으로 변경하였습니다.

또한 사이트 배너의 배경 이미지를 여러 게임의 대표 이미지들이 시간 경과에 따라 페이드되며 나타나는 식으로 제작하는 데 AI를 활용하였습니다.
